This directory stores frozen verification suites and nothing disposable.

`v1_gold/` is the current oracle-backed reference suite for the brute-force baseline. These files are generated by `V1`, then frozen and reused to validate all later implementations.
`v1_gold_manifest.json` stores the suite audit result and deterministic SHA256 hashes for the frozen files.

Each testcase records:
- the input points
- the expected peel order
- the expected per-step area decreases
- the expected hull snapshots
- metadata such as seed, point count, grid limit, and epsilon settings

The current fixed suite contains:
- `900` cases with `25` points
- `100` cases with `100` points
- points generated on a grid with coordinates in `[-1_000_000, 1_000_000]`

Temporary run outputs do not belong here. Verification artifacts are written under `agent/temp_files/`.
